 Form validation with error messages
 Semantic HTML and consistent styling

##  Built With
 HTML5
 CSS3 (Flexbox, Grid, Media Queries)
 JavaScript
 LocalStorage

##  Pages
 **index.html** - Home with featured dishes
 **menu.html** - Complete menu
 **add-dish.html** - Form to add new dishes

## How to Run
1. Clone repo: `git clone https://github.com/yourusername/bangola-kingdom`
2. Open `index.html` in any modern browser
3. Navigate using the menu
4. Add your dish of choice

## Data Persistence
All community dishes are saved in browser localStorage using the key `bangola-community-dishes`.
